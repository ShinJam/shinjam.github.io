## [평범한 배낭](https://www.acmicpc.net/problem/12865)

### SOL1
```python
N = int(input())
W, DP = [], []
for _ in range(N):
    W.append(list(map(int, input().split())))
    DP.append([0]*(1<<N))
    
def get_weight(last,visited):
    if visited == (1<<N) - 1:  
        return W[last][0] or inf  
 
    if DP[last][visited]:  
        return DP[last][visited]
```

- TPS(Travel Salesman Problem) 유명한 문제로 아직까지도 연구중인 NP문제
