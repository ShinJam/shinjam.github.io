# 새 | [1568](https://www.acmicpc.net/problem/1568)

### 분류
`탐색`

### 입력
첫째 줄 = 새의 수 N -> $10^9$이하

### 출력
모든 새가 날아갈 때까지 걸리는 시간 -> K마리 새가 한 번 날 때 1초

- ***sol 1***

```python
N = int(input())
ans = 0
K = 1

while N > 0:
    if K > N:
        K = 1
    ans += 1
    N -= K
    K += 1
    
print(ans)
```

### Notes
K 숫자 만큼 K 마리 새가 N에서 없어 지기 때문에, 알고리즘 시간 복잡도는 O($\log(N)$)다.
